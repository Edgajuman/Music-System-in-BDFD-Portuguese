
# Sistema de música no BDFD v7.0 pós-BETA (Privado) Português

Um sistema avançado para reproduzir música com o seu bot Discord no Bot Designer For Discord.
**ESTE SISTEMA FOI REDESENHADO PARA ECONOMIZAR MAIS RECURSOS, SER MAIS FÁCIL DE USAR E SER COMPATÍVEL COM AS NOVAS LIMITAÇÕES DO BDFD**
> Para solicitar acesso a esta nova versão, inicie sessão no servidor de suporte, abra um ticket com o motivo Suporte ou Outro e indique-nos a razão pela qual deseja o sistema de música para o seu bot BDFD.

## Configuração

#### Só precisa de criar 1 variável obrigatória


| nome | valor     |
| :-------- | :------- |
| `TOKEN` | `O token do seu bot Discord` |

## Comandos
- ! (Este código tem 6 comandos em um, então basta inserir o seu prefixo, por exemplo: !)
- !play
- $onInteraction[Musics]

## O que contém?
> Por enquanto, apenas:
- Reprodução de música
- Sistema de pausa e retomada
- Sistema de avançar e retroceder
- Sistema de salto para avançar para qualquer música na fila (máximo de 25)
- Função de parar para interromper a música e desligar o bot

## Possíveis bugs
O sistema está totalmente funcional, embora tenha alguns bugs que serão corrigidos posteriormente.
Alguns bugs podem incluir:
- Pode demorar algum tempo para processar a música, principalmente devido ao tamanho e duração do vídeo/música que está a ser pesquisado.
- Quando a fila de música termina, pode não repetir em loop. Isso será corrigido em atualizações futuras.
